# UnityOneWeek_CoinCat_CodeSample

Unity1Week参加作品「つながれ!コインキャット!」の、自作コードを抜粋したポートフォリオ用リポジトリです。

## 作品概要

「つながれ!コインキャット!」は、決められたお題に沿って1週間で制作・公開したUnity製ゲームです。

短い制作期間の中で、ゲームルールの設計、Unity / C#による実装、UI調整、公開までを一貫して行いました。

作品URL：  
https://unityroom.com/games/coincatconnect

## 掲載内容

### Game

ゲーム本編に関するコードです。

- アバター制御
- コイン制御
- ゲーム進行管理
- UI制御
- 背景演出
- クリア演出

### Title

タイトル画面に関するコードです。

- タイトル画面の進行管理
- ロゴ演出
- 隠し効果音機能

### Extensions

個人制作で使用している自作の拡張メソッドです。

- 可視判定
- ランダム処理
- Vector関連処理

### Editor

Unity Editor上での作業を補助する自作エディタ拡張です。

## 補足

本リポジトリは、採用選考においてコードの書き方や実装方針を確認していただくことを目的として、自作コードの一部を抜粋して公開しています。

素材、音楽、画像、外部アセット、`.meta`ファイル等は含めていないため、このリポジトリ単体ではUnityプロジェクトとして動作しません。

実際のゲームは、上記のunityroomリンクから確認できます。
